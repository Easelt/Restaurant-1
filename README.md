# Broaster - Classic Chicken Fast Food

A classic, elegant website for Broaster, a chicken fast food restaurant.

## Features

- Classic, timeless design
- Responsive layout
- Smooth scrolling navigation
- Menu showcase
- About and contact sections

## Getting Started

Simply open `index.html` in your web browser to view the website.

## Files

- `index.html` - Main HTML structure
- `styles.css` - Classic styling and design
- `script.js` - Interactive features and smooth scrolling

## Design

The website features a classic design with:
- Warm brown and gold color scheme
- Traditional serif typography
- Elegant gradients and shadows
- Classic fast food restaurant aesthetic
